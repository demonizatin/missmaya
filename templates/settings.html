AWS Bedrock

Two ways to authenticate. Use either the bearer token or the IAM credentials path — not both.

About the bedrock-api-key-… token you have: that's an AWS Bedrock API key. It's actually a presigned URL packaged as a bearer token; the Anthropic SDK passes it via the AWS_BEARER_TOKEN_BEDROCK env var. Heads up: these tokens expire (yours has X-Amz-Expires=43200 = 12 hours). After expiry you'll need to refresh it, or switch to the IAM credentials path below for stable auth.
